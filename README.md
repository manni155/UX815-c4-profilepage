# Challenge #4: Profile Page
Design a profile page for a UX-focused social network. What information would you display? How would it look? Create a design in Figma and then build out the page in HTML and CSS. Export the design from Figma as a PNG and include it in your GitHub repository. Commit and publish your design to Netlify and upload the public Netlify and GitHub URLs to the C4: Dropbox.

Tip: don't overthink (or over-do) the Figma design. Go for simple--keeping in mind that you have to build this out in HTML and CSS. The goal is not complexity, it's gaining practice translating the design from Figma to code.
